What is Shnip.it?

Shnip.it is a solution we came up with to the problem of transferring snippets of code between projects. Sick of having to type that connection to a database for every website? Save it as a snippet. Have a series of HTML/CSS templates for page layouts? Save them as snippets! Whatever you want to reuse the code for, shnip.it is your answer: Portable, Quick and Clean.
